# EARLIER

## 注释:
返回当前行中参数列的值


## 语法
EARLIER(列)

## 参数
当前表的列

## 返回
值，当前行和参数列交叉的单元格的值

## 示例
### 计算当前订单产品的目前累计销量


```
sumx(
    filter(
        '订单表',
        '订单表'[序号]<=EARLIER('订单表'[序号]
        &&'订单表'[产品名称]=EARLIER('订单表'[产品名称])
        ),
    '订单表'[销售数量]
)
```

## 说明

EARLIER函数主要用于计算列，获取当前行上下文。因为它需要扫描每一行，计算性能相对较慢
